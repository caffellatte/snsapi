# snsapi
API wrapper for snscrape

NestJS education project

## snscrape Installation

```bash
# Clone repository
git clone https://github.com/JustAnotherArchivist/snscrape
```

```bash
# Install with pip
python3 -m pip install .
```

## Nest Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
